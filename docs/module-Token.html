(inner) stockerJWT(jwt, cle)

Stocke un JWT (JSON Web Token) dans la base de données avec la clé secrète correspondante. On conserve les token dans la bdd car une clé secrete est générée à chaque redémarrage du serveur. Ceci rend l'utilisation des anciens token inutilisables. On stocke donc les token avec leur clé afin qu'ils soient exploitables.
